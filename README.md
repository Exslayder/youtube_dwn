# Скрипт для скачивания с Youtube

Консольная утилита для скачивания видео и аудио с YouTube.
Для использования через ".exe", в релизе v1.0.0 доступен архив для скачивания файла.


## Возможности
- Скачать видео
- Скачать аудиодорожку
- Сохранение настроек

## Запуск
```bash
python main.py
```
